# Hello World

Welcome to my blog! This is the first post, written in Markdown.

## Why a Blog?

I created this space to:

- **Share knowledge** — things I learn along the way
- **Improve writing** — explaining ideas clearly is a skill
- **Build in public** — document projects and progress

## What to Expect

I'll write about programming, tools, workflows, and occasionally other interests. Posts will be practical and to the point.

## Tech Stack

This blog is built with:

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Giscus](https://giscus.app) — comments via GitHub Discussions
- [Pagefind](https://pagefind.app) — static search
- Deployed on [Vercel](https://vercel.com)

## Code Example

Since I'm a developer, here's a quick TypeScript snippet:

```typescript

  return `Hello, ${name}! Welcome to my blog.`;
```
